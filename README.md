## 📖 Overview

This project implements a **Handwritten Digit Recognition System** using **Logistic Regression**. The model is trained on the **MNIST dataset** to recognize handwritten digits (0–9).

The project demonstrates a complete machine learning workflow, from data preprocessing to model evaluation and prediction.

---

## ✨ Features

- 📊 Train a Logistic Regression classifier
- 🔢 Classify handwritten digits (0–9)
- 📈 Evaluate model performance
- 📉 Generate a Confusion Matrix
- 📋 Classification Report
- 🎯 Predict unseen handwritten digits
